Core Capabilities

Single‑Plane Balancing

Designed for “thin” rotors where correction occurs in one plane. Uses the Influence Coefficient method with vector math to calculate precise correction weights.

Includes polar plots, zoom/pan visualization, and optional Least‑Squares optimization.

Two‑Plane Balancing

For wider rotors requiring simultaneous correction in two planes. Uses a full 2×2 influence matrix with clear cross‑effect visualization.

Provides vector diagrams and a structured workflow for consistent, repeatable results.

Trial Weight Estimator

Helps determine an appropriate trial weight using machine response assumptions and rules of thumb.

Improves first‑try success rates and reduces wasted runs.

4‑Run No‑Phase Method

Phase‑free balancing using vibration amplitudes. Uses trilateration and optimization to locate the unbalance vector.

Allows flexible trial weight placement—not limited to typical 120° patterns seen in other tools.

Weight Manager

Consolidate existing weights into one location or split weights for optimal balancing results.

Keeps the rotor clean, organized, and easier to service in the future.

Severity Screening

Evaluates vibration severity using an ISO 10816‑inspired hybrid interpretation with color‑coded thresholds.

Quickly understand existing vibration levels before and after balancing.
